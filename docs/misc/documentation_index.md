# Documentation Index

This folder centralizes operational notes, design references, and onboarding guides for the scraper platform. Key entry points:

- **misc/root_overview.md** – High-level repository overview relocated from the root.
- **architecture/** – System design and component interactions.
- **requirements/** – Business, technical, and release-specific requirement sets.
- **workflows/** – Common operational runbooks and integration guides.
- **onboarding/** – Standards for adding new scrapers and configuring environments.
- **troubleshooting/** – Debugging playbooks, validation checks, and error references.
- **changelogs/** – Release notes, migration summaries, and critical fixes.
- **misc/** – Cross-cutting references such as glossary, roadmap, FAQs, and status snapshots.

Remove or update documents here as features evolve to keep this directory concise.
