# Production Readiness

Checklist for confirming production readiness of the scraper platform.

## Checklist
- [ ] Database connectivity validated with required credentials.
- [ ] Optional dependencies installed where needed (`psycopg2`, `selenium`).
- [ ] Monitoring/logging configured for Airflow and agent runs.
- [ ] Pipeline configurations reviewed for target sources.

Update this checklist as environments move through staging to production.
